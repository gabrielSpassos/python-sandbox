# Websocket POC

POC to test websockets lib and basics protocols on web sockets

### Usage

* Configure venv

```bash
python3 -m venv .venv
source .venv/bin/activate
```

* Install dependencies

```bash
pip3 install -r requirements.txt
```

* Run server
```bash
python3 src/server.py
```

* Run client
```bash
python3 src/client.py
```

* Deactivate venv
```bash
deactivate
```
