# CLI POC

### Usage

* Create virtual env
```
python3 -m venv .venv
. .venv/bin/activate
```

* Deactivate virtual env
```
deactivate
```

* Install dependencies
```
pip3 install -r requirements.txt
```

* Install executable
```
pip3 install -e .
```

* Run executable
```
hours
```

* Run Manually
```
python3 src/hours/main.py
```
