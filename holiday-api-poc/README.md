# Holiday API POC

> https://www.abstractapi.com/api/holidays-api

### Usage

* Create virtual env
```
python3 -m venv holiday_api_venv
source holiday_api_venv/bin/activate
```

* Install dependencies
```
pip3 install -r requirements.txt
```

* Run
```
python3 src/main.py
```
